Changement de paramètre

Vous écrivez directement une propiété CSS affilié à certaine partie des pages.

Vous pouvez donc pour les couleurs utiliser:

  • le code hexa corespondant (exemple #AAAAAA)
  • les fonctions rgb et rgba
  • mettre le nom d'une couleur connu du Browser

Pour les tailles de caractères vous avez les valeurs en unité : px,em,in,pt,pc,cm,mm.

Si vous enregistrer un chamgement de paramètre il sera effectué aussitot mais rechagera les pages et vous perdrez votre avancement
si vous avez une partie avec une fenêtre pour l'arbre ouvert celle-ci sera uniquement fermé.

En Jeu

Vous pouvez utiliser le click gauche pour vous balader dans l'arbre et changer de formule courante.

Vous pouvez avoir une fenêtre dédier à l'arbre.
